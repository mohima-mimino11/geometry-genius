1.What is DOM? What is the purpose of DOM?

Answer: Dom or document object model is logical structure of how elements are designed in html. The purpose behind dom is maintain a standard for setting and manipulating tags in html also for handling events of any website.

2.How can you select element from dom?

  • document.getElementsByTagName('tag-name') ==> returns the tag(ex: div, h1, li, p) you put in the paranthesis Also returns HTMLCollection which is a array like object but not an array.
  • document.getElementsByClassName('class-name') ==> returns the tags who share the same class name inside html tags. unlike previouse selection technique it also returns HTMLCollection which is a array like object but not an array.
  • document.getElementById('id-name') ==> returns only the tag which have the exact tag which is put inside the paranthesis. It only returns one element as we always give unique id name in html tags.
  • document.querySelector('specific-tag') ==> returns only the tag you specified (eg: like for classname you need give .classname in the paranthesis).
  • document.querySelectorAll('tag-name') ==> returns the tag(ex: div, h1, li, p) you put in the paranthesis Also returns HTMLCollection which is a array like object but not an array.

3.What is event bubble?

Answer: In some nested elements When we need to get the topmost tag or parent tag by selecting the lowest tag or child tag but both have registered listener for event handling that's when we use event bubbling. For example say a section have a ul tag which have few li inside that when we click on li it then select li then ul then section.

Answer: Calback functions are those functions which are function called from a outsider function. The major advantage of using callback functions is it's reusability.
